# Preparing Dietary Data
Polyphenol Estimator can work with the following diet input files:
 - ASA24 Recall or Records Items Files
 - WWEIA, NHANES "Individual Foods" Recall Files
    
ASA24 files can be pulled off the ASA24 Researcher Website, with the Items File required for running Polyphenol Estimator. If you are working with AMPM data from NHANES, you will need to prepare your data in a specific format. The following sections provide information on how you can pull down your ASA24 data and prepare NHANES dietary data.
